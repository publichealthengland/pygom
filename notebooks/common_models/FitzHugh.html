FitzHugh

FitzHugh#

FitzHugh() - the FitzHugh4 model without external stimulus

This is a commonly used model when developing new methodology with regards to ODEs, see 11 and 5.

#TODO why common model?

\[\begin{split}\begin{aligned} \frac{dV}{dt} &= c ( V - \frac{V^{3}}{3} + R) \\ \frac{dR}{dt} &= -\frac{1}{c}(V - a + bR). \end{aligned}\end{split}\]

An example of using this model follows.

import numpy

from pygom import common_models

import matplotlib.pyplot as plt

ode = common_models.FitzHugh({'a':0.2, 'b':0.2, 'c':3.0})

t = numpy.linspace(0, 20, 101)

x0 = [1.0, -1.0]

ode.initial_values = (x0, t[0])

solution = ode.integrate(t[1::])
